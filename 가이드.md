# 📖 영림 매장 자동화 시스템 설치 및 실행 가이드 (V8.1)

이 가이드는 다른 컴퓨터에서 프로그램을 처음 설치하고 실행하는 과정을 설명합니다.

---

## 1️⃣ 사전 준비 사항

- **Python 설치**: Python 3.10 이상의 버전이 필요합니다.
  1. [Python 공식 다운로드 페이지](https://www.python.org/downloads/windows/)에 접속합니다.
  2. **Python 3.11.x** (또는 최신 정식 버전)의 `64-bit installer`를 다운로드하여 실행합니다.
  3. 설치 창 하단의 **'Add Python to PATH'** 옵션을 반드시 체크한 후 `Install Now`를 클릭하세요.
- **브라우저**: **Avast Secure Browser**가 설치되어 있어야 합니다. (기본 설치 경로: `C:\Program Files\AVAST Software\Browser\Application\AvastBrowser.exe`)

---

## 2️⃣ 프로그램 다운로드 및 설정

1. **코드 내려받기**: 
   - GitHub 저장소에서 `Code` -> `Download ZIP`을 눌러 압축을 풀거나,
   - 터미널에서 `git clone https://github.com/kangHo-Jun/Shop_Automation.git` 명령어를 실행합니다.

2. **환경 설정 (.env)**:
   - 폴더 내의 `.env.example` 파일을 복사하여 `.env`로 이름을 바꿉니다.
   - 필요한 경우 메모장으로 열어 포트 번호나 대기 시간 등을 수정합니다. (기본 설정 사용 권장)

---

## 3️⃣ 가상 환경 및 패키지 설치 (자동화)

프로그램 폴더 내의 **`setup_env.bat`** 파일을 더블 클릭하여 실행하면 다음 과정이 자동으로 진행됩니다:
- 가상 환경(`.venv`) 생성
- `.env.example`을 복사하여 `.env` 생성 (없을 경우)
- 필수 패키지(`requirements.txt`) 자동 설치

---

## 4️⃣ 프로그램 실행

설치가 완료되면 폴더 내의 실행 파일을 사용하여 프로그램을 시작할 수 있습니다.

1. **`run_v8_server.bat` 실행**: 파일을 더블 클릭하면 서버가 시작됩니다.
2. **대시보드 접속**: 웹 브라우저를 열고 [http://localhost:5080](http://localhost:5080) 에 접속합니다.
3. **사용 시작**: 대시보드에서 실시간 상태를 확인하고 업로드 작업을 진행합니다.

---

## ⚠️ 주의 사항

- **로그인 상태**: 프로그램을 실행하기 전, Avast Browser에서 영림(OMS) 사이트에 한 번 로그인하여 세션을 유지해 두는 것이 좋습니다.
- **포트 충돌**: 만약 5080 포트가 이미 사용 중이라면 `.env` 파일에서 `FLASK_PORT`를 다른 번호로 변경하세요.
- **로그 확인**: 에러가 발생할 경우 `logs/` 폴더 내의 최근 로그 파일을 확인하여 원인을 파악할 수 있습니다.
